# FIA Hackathon Landing Page

A premium, interactive, and fully responsive hackathon landing page built with **Semantic HTML5**, **CSS3**, and **Vanilla JavaScript**.

## Features
- Premium dark-mode design with glassmorphism
- CSS3 animations (floating astronaut, orbital rings, scroll reveals)
- Fully responsive (mobile-first)
- Zero dependencies — no build tools required

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript

## Getting Started
Open index.html in any browser. No server needed.
